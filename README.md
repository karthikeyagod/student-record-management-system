# Student Record Management System

A console-based application designed to manage student records using file handling.

## Features
- Add student records
- Display all records
- Search student by roll number
- Delete student records
- Persistent storage using files

## Technologies Used
- Java
- File Handling
- Object Oriented Programming

## Purpose
This project was developed to strengthen understanding of file handling,
data structures, and menu-driven program design.
